# 流浪图书管理系统（Wandering Library System）

这是一个基于 Python 编写的命令行图书管理系统，旨在模拟一个小型图书馆的借阅流程。每本沉默的好书都是一座流浪的岛屿，希望你有缘发现并着陆，为精神家园找到一片栖息地。

## 📌 功能介绍

- 📖 查询所有书籍
- ➕ 添加新书籍
- 📚 借阅书籍
- 🔁 归还书籍
- ❌ 退出系统

## 🧠 技术实现

- 使用类 `Book` 表示一本书的属性（书名、作者、推荐语、借出状态）
- 使用类 `BookManager` 管理书籍列表与交互菜单
- 支持命令行交互，实时录入与修改数据
